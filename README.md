# Online Groceries Store App with MVVM in SwiftUI

I have built an advanced Online Grocery App for iOS using MVVM (Model-View-ViewModel) architecture. SwiftUI is Apple's modern declarative framework for building user interfaces and MVVM ensures a clean and organized codebase. In this project, I have created a powerful and user-friendly app that allows users to shop for groceries online, add items to the cart and complete their purchases seamlessly.
